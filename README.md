# StomaScope – Crop Disease Detector

StomaScope is an AI-based crop disease detection system that identifies plant diseases from leaf images using deep learning.

## Features
- Upload a leaf image
- Predict crop disease using CNN model
- Display prediction confidence
- Grad-CAM visualization to highlight infected regions

## Technologies Used
- Python
- TensorFlow / Keras
- Streamlit
- OpenCV
- NumPy

## Model Details
The model was trained on a plant disease dataset and can classify multiple crop diseases from leaf images.

## How It Works
1. User uploads a leaf image.
2. The image is processed and passed to the trained CNN model.
3. The model predicts the disease class.
4. Grad-CAM generates a heatmap showing infected regions.

## Deployment
The application is deployed using Streamlit.
